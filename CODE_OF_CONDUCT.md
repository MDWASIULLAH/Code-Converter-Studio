# Code Of Conduct

## Our Standard

This project welcomes helpful, respectful, and constructive participation.

Examples of positive behavior:

- Giving clear feedback.
- Helping others learn.
- Being patient with beginner questions.
- Focusing on the work instead of personal attacks.

Examples of unacceptable behavior:

- Harassment or insulting comments.
- Publishing private information.
- Repeated disruptive behavior.
- Discriminatory language.

## Enforcement

Project maintainers may remove comments, reject contributions, or block participants who do not follow this code of conduct.
