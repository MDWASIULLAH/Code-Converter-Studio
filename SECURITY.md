# Security

## Local Code Execution

The browser-only features run inside the browser preview. The optional local runner can execute Python, Java, C, and C++ code on your machine through `server.js`.

Only run code that you trust.

## Reporting Issues

If you find a security problem, please avoid posting exploit details publicly. Open a private report or contact the repository owner.

## Recommendations

- Do not expose the local runner to the public internet.
- Keep the server bound to `127.0.0.1`.
- Do not paste or run secrets inside the editor.
- Review code before running it locally.
